This repository was generated from the LC2A presentation. It includes a responsive Tailwind-based multi-page static site, a slider (Swiper), and AOS animations.

## How to use
1. Replace images in `assets/images/` with the official logo and photos extracted from the presentation (already attempted extraction).
2. Update the contact form action in `contact.html` (Formspree or Netlify Forms).
3. Test locally by opening `index.html` in a browser, or serve with a static server like `live-server`.

## Deploy
- Push to GitHub and enable GitHub Pages on the repository (Settings > Pages).
- Or deploy to Netlify / Vercel by connecting the repository.
